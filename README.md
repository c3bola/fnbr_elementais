# 🎮 Coleção de elementais do Fortnite

Olá, pessoal! Sejam muito bem-vindos ao **coleção de elementais do Fortnite**. 

Este projeto é uma adaptação e tradução completa para Português do Brasil (PT-BR) a partir do repositório original [fnsprites](https://github.com/staticvacant/fnsprites.git). O objetivo é simples: dar a você uma ferramenta leve e visual para catalogar seus elementais, acompanhar seu progresso e dominar cada um deles no jogo!

## ✨ Funcionalidades

* 🔍 **Pesquisa e Filtros:** Encontre rapidamente o elemental que você procura por nome ou categoria (Básico, Dourado, Gelatina, Galáxia, Gema, Holofoil e Fenda).
* 📊 **Acompanhamento em Tempo Real:** Veja exatamente a porcentagem da sua coleção e o nível de domínio (*maestria*) sobre os elementais, tudo com barras de progresso dinâmicas.
* 🖼️ **Exportação de Imagens:** Gere imagens automáticas da sua coleção — sejam os itens que você já tem, os que faltam, os que ainda não dominou ou os seus favoritos. Perfeito para compartilhar com os amigos!
* 🔗 **Modo Compartilhamento:** Você consegue gerar um link único para exibir sua coleção para seus colegas.
* 📱 **100% Responsivo:** Acompanhe seu progresso e marque seus itens tanto no PC quanto no celular, sem complicações.

## 🛠️ Como Utilizar

Como se trata de uma aplicação web estática, você pode executá-la diretamente no seu navegador:

1. Baixe os arquivos do repositório ou clone o projeto.
2. Abra o arquivo `index.html` no seu navegador favorito (Chrome, Edge, Firefox, etc.).
3. Clique nos blocos (cards) dos elementais para marcá-los como **Possuídos**, e use a coroa para marcá-los como **Dominados**!

*(Dica: Você pode hospedar o projeto facilmente no GitHub Pages, Vercel ou Netlify para acessar de qualquer lugar).*
